## Usage

- Run frontend

```bash
git clone https://github.com/AlexMubarakshin/al-blog.git
cd ./al-blog/frontend
npm i
npm start
```

- Run backend

```bash
cd ./al-blog/backend

# Rename `.env.example` to .env and paste your params.

$ npm i
$ npm start

# Visit localhost:3000
```
